UI Components / Snail Trail

The Snail Trail component can be used as a supplementary navigation bar, that also functions as a carousel on mobile devices. The default Snail Trail supports one required and two optional properties:

name: a required string specifying the name of the Snail Trail, displayed as the item.

src: a url string for a JSON object that contains data to generate the component.

env: a string specifying the environment the component should be built in (Int, Demo, or Prod)

Note: If src and env aren't included, the source is defaulted to Prod.
