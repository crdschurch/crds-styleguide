UI Components / Shared-footer

The default shared-footer supports two optional properties (src): a url string for a JSON object that contains data to generate the component, and (env): a string specifying the environment the component should be built in. (env) can be one of the following strings: ("Int", "Demo", "Prod"). If neither properties are included, the source is defaulted to Prod.
