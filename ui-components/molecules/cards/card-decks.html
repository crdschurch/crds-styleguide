Card Decks

A card deck is a group of cards which share the same height as the tallest card in the deck. Height is determined by the length of content within a card, and the card with the most content will dictate the height of the other cards in the deck.

The examples below pull the carousel effect by using the crds-card-carousel shared library, along with the library's dependencies..

Default Card Deck

Adding a data attribute of data-crds-carousel="mobile-scroll" to a card deck and instantiating the crds-card-carousel script enables side-scrolling on mobile-sized screens only. On larger screens, the layout will be an expanded view, creating new rows for each set of four cards.

Card Deck Carousel

Adding a data attribute of data-crds-carousel="always-scroll" to a card deck and instantiating the crds-card-carousel-script will create a side-scrolling effect on all screen sizes.
